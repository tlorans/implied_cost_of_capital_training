Exercise 1: Basic Calculation
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Given the following information:

* Current stock price: $50
* Expected dividend next year: $2
* Growth rate: 5%

Calculate the implied cost of capital using the Gordon Growth Model.

**Solution:**

.. math::

   r = \\frac{\\$2}{\\$50} + 0.05 = 0.04 + 0.05 = 0.09 = 9\\%
